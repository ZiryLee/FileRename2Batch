 **《Ziry批量修改图片名V1.0》** 


这个程序是用JAVA开发的，所以你操作系统需要安装好JAVA环境，

如果没有安装JAVA环境则请下载jre8_86.zip，解压安装jar8_86.exe，

按指示安装完成后，双击运行：[Ziry批量修改图片名V1.1.jar](https://git.oschina.net/zirylee/FileRename2Batch/attach_files)
